# Supabase - Google Sheets Sync

Real-time two-way synchronization between Supabase and Google Sheets.

## Setup

1. **Deploy to Vercel:**
   - Fork this repository
   - Go to [vercel.com](https://vercel.com)
   - Import your GitHub repository
   - Deploy

2. **Configure Google Sheets:**
   - Open your deployed app
   - Enter your Google Sheet URL and Sheet Name
   - Save configuration

3. **Setup Google Apps Script:**
   - Go to [script.google.com](https://script.google.com)
   - Create new project
   - Paste the Google Apps Script code
   - Set up trigger for `onEdit` function

## Features

- ✅ Real-time sync from Google Sheets to Supabase
- ✅ Manual sync controls
- ✅ Bookings and Invoices management
- ✅ Mobile responsive

## Technology Stack

- Frontend: HTML, CSS, JavaScript
- Database: Supabase
- Deployment: Vercel
- Automation: Google Apps Script
